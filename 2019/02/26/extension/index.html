注意点

1. content_scripts 中 css 文件会直接插入到页面，要注意否则影响到所有页面原来的样式。
2. options 页面 input 框中显示 url 形式的字符串 ，在最后会默认加一个 / ，如果是空字符串 ‘’ 是也会显示 / 所以在保存输入的 url 时要去掉最后的 / 。
