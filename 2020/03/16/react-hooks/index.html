hooks api 类型

Hooks 主要分为以下三种：

  • State hooks： 可以让函数式组件使用 state
  • Effect hooks： 可以让函数式组件使用生命周期和 side effect， 如 useEffect 可以让组件具备 life-cycles。
  • Custom hooks： 自定义 hooks
